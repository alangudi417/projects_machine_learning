English:
This project focuses on building a Machine Learning model to recommend the most suitable mobile plan—Smart or Ultra—for Megaline customers. Using behavioral data from subscribers who have already migrated to the new plans, the goal is to analyze usage patterns and train a classification model capable of predicting the optimal plan for new users.

The data was preprocessed beforehand, allowing this project to focus directly on model development, training, and evaluation. The objective is to achieve the highest possible accuracy, with a minimum required threshold of 0.75.

Español:
Este proyecto se centra en desarrollar un modelo de Machine Learning capaz de recomendar el plan móvil más adecuado para los clientes de Megaline: Smart o Ultra. Utilizando datos de comportamiento de suscriptores que ya han migrado a los nuevos planes, el objetivo es analizar sus patrones de uso y entrenar un modelo de clasificación que pueda predecir el plan óptimo para nuevos usuarios.

Los datos fueron preprocesados previamente, lo que permite enfocarse directamente en la construcción, entrenamiento y evaluación del modelo. La meta es lograr la mayor exactitud posible, con un umbral mínimo establecido de 0.75.
